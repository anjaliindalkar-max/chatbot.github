# chatbot.github
CKT College Student Help Desk Chatbot

A simple desktop-based chatbot application built using Python and Tkinter. This project acts as a virtual student help desk for CKT College, providing quick information related to courses, admissions, fees, scholarships, college timings, and contact details.

Features

- Interactive graphical user interface (GUI)
- Instant responses to student queries
- Information about:
  - Available courses
  - Admission process
  - Course fees
  - Contact details
  - College timings
  - Scholarships
- Quick access buttons for common queries
- Timestamped chat messages
- Enter key support for sending messages

Technologies Used

- Python 3
- Tkinter
- tkinter.scrolledtext
- datetime module

Installation

1. Clone the repository:

git clone https://github.com/your-username/CKT-College-Student-Help-Desk.git

2. Navigate to the project directory:

cd CKT-College-Student-Help-Desk

3. Run the application:

python chatbot.py

Usage

- Launch the application.
- Type your query in the input box and click Send or press Enter.
- Use the quick-access buttons for frequently asked questions.
- The chatbot will display relevant information in the chat window.

Supported Queries

Examples of questions the chatbot can answer:

- "What courses are available?"
- "Tell me about admissions."
- "What are the course fees?"
- "How can I contact the college?"
- "What are the college timings?"
- "Do you offer scholarships?"
- "Help"
